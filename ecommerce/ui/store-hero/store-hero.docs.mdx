## Hero

A here component for the Store

### Component usage

```js
<StoreHero
  headingText="store heading"
  linkText="click me"
  href="/"
  background="blue"
/>
```

### Using props to customize the text

Modify the text to see it change live:

```js live
<StoreHero
  headingText="store heading"
  linkText="click me"
  href="/"
  background="blue"
/>
```
